# rs-485-relay
## Управление релейным модулем через интерфейс RS-485 по протоколу Modbus RTU

### Схема подключения

```
[PC USB <-> RS-485]            [RS-485 relay module]
[A+]  >----------------------< [A+]
[B-]  >----------------------< [B-]
[GND] >----------------------< [GND]
                               [DC 7 - 24v] >---------< [Power supply]
                               [out-1]
                               [out-2]
                               [out-3]
                               [out-4]
```
